# Максим Громов — Персональный тренер

Профессиональный лендинг для персонального фитнес-тренера.

## Технологии
- HTML5 / CSS3 / JavaScript (Vanilla)
- Без сторонних библиотек и фреймворков
- Mobile-first адаптивный дизайн

## Возможности
- Эффект печатающегося текста
- Параллакс в Hero секции
- Анимации при скролле (Intersection Observer)
- Счётчики с анимацией
- Валидация формы с маской телефона
- Burger-меню для мобильных
- Прогресс-бар страницы
- Tilt-эффект на карточках (desktop)
- Glitch-эффект на логотипе
- Ripple-эффект на кнопках

## Запуск
Открыть `index.html` в браузере или использовать Live Server в VS Code.
